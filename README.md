# AutoPointer
音声操作が可能なWebページ


## 作成物の説明
PCなど、デジタルデバイスの操作が苦手なユーザー、主に高齢者などを対象に何をしたいかを伝えるだけで操作が可能なWebページを作成した。これによりユーザーは○○をするために、～の操作を行うといった実行ステップを脳内で構築する必要がなくなり、実行の淵の問題を解消できる。より直感的な操作を実現するために音声認識技術を用いている。
また、このWebページでの体験を通して、○○を行うために～の操作を行うという実行ステップを構築できるように、音声操作によって突然画面が変わったりするのではなく、大きなカーソルが動き、どのような操作を行っているのかを視覚的に理解できるように工夫した。これにより、実行の淵のみならず、評価の淵の問題も解消できる。

## 担当役割
アイデアの作成、企画書の作成は自らで行い、要件定義はAIとの共同で作成した。実装はAIエージェントに行わせた。途中、部分的な修正などは自らで行ったが、大部分はAIによるコーディングである。

## 課題と解決方法
当初は軽量なSLMをローカルで実行し、音声操作を完全に理解するシステムの作成を目指したが、Webブラウザの動作が重くなったり、解析に都度差が生じたりしたため、本Webページが軽量かつ簡易なものであることを利用して、特定のテキストが操作指示文(操作指示文は音声認識で得られたテキストを前処理したもの)に含まれていた場合に、対応する操作を実行するように変更した。これによって、安定かつ軽量な動作が保たれた。

## 使用技術
HTML5, CSS3, JavaScript, SpeechRecognition, SpeechSynthesis, Transformers.js, ONNX Runtime Web / WebGPU, Gemma 4 E2B, Web Workers, MutationObserver, requestAnimationFrame, SQLite, FastAPI
